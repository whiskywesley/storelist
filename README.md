# jquery.storelist.js
this is a tiny plugin for showing lists in xml with Jquery. <br />
such like branch stores list.

> usage
basic:
- select by element:
```
$('div').storelist('yourXmlPath');
``` 

- select by class/id:
$('.class').storelist('yourXmlPath'); 
$('#id').storelist('yourXmlPath');
